# JMA API 出力JSON形式の仕様

本ドキュメントでは、JMA API (`/jma/latest`) が出力するJSON形式について解説します。
この形式は、特定の表示端末やVRChatなどの外部アプリケーションでの利用を想定しています。

## 概要
APIは常に**オブジェクトの配列**を返却します。情報がない、または安定している場合は「安定」状態を示すオブジェクトを一つ含む配列を返します。

### 保持・消去ポリシー
各情報は、サーバーが取得してから**10分間**キャッシュに保持されます。10分経過した情報は自動的に配列から除外されます。また、サーバー起動時に10分以上前の古い情報がフィードに残っていた場合、それらは処理されません。

## 共通フィールド
| フィールド | 型 | 説明 |
| :--- | :--- | :--- |
| `type` | `String` | 内容の種類（表示の優先度や色分けに使用） |
| `id` | `String/Number` | 情報の一意なID (EventIDなど) |
| `lines` | `Array<Object>` | 表示用オブジェクトの配列。下記の「lines フィールドの詳細」参照 |
| `timestamp` | `String` | 情報が処理された日時 (ISO 8601形式) |

---

## 種類別詳細 (`type`)

### 1. 地震情報 (`type`: `earthquake_*` / `emergency`)
震度に応じた種別が設定されます。
- `earthquake_1` : 最大震度 1
- `earthquake_2` : 最大震度 2
- `earthquake_3` : 最大震度 3
- `earthquake_4` : 最大震度 4
- `emergency` : 最大震度 5弱 以上、震度速報、または**緊急地震速報**
- `earthquake` : 震源情報のみ（震度情報がない場合）

#### 緊急地震速報（EEW）の特記事項
緊急地震速報の場合、`lines` 配列の要素は**常に1つ**になります。
テキストにはタイトル、見出し、対象地域が `\n` で結合された状態で格納されます。
- 予報の場合: `【緊急地震情報（予報）】`
- 警報の場合: `【緊急地震情報（警報）】`

---

## `lines` フィールドの詳細
各要素は以下の形式のオブジェクトです。
```json
{
  "text": "表示するテキスト内容",
  "duration": 3.0
}
```

| フィールド | 型 | 説明 |
| :--- | :--- | :--- |
| `text` | `String` | 表示用のテキスト行。リッチテキストタグ（`<align>`, `<color>`, `<nobr>`等）を含む場合があります。 |
| `duration` | `Number` | その行の表示推奨秒数。 |

### 表示秒数 (`duration`) の目安
- タイトル行: 1.5秒
- 標準的な情報行: 3.0秒
- 震度などの詳細行: 4.0秒
- 緊急地震速報（詰め込み形式）: 15.0秒

---

## 出力例 (JSON)

### 通常の地震情報
```json
{
  "type": "earthquake_2",
  "id": 20260402162719,
  "lines": [
    { "text": "<align=\"center\">地震情報", "duration": 1.5 },
    { "text": "午後4時27分ごろ 地震がありました", "duration": 3.0 },
    { "text": "震源は十勝地方中部\n深さ70キロ マグニチュード3.2", "duration": 4.0 },
    { "text": "<u>震度2</u> <indent=5em><nobr>浦幌町</nobr></indent>", "duration": 3.0 }
  ],
  "timestamp": "2026-04-02T09:14:09.441Z"
}
```

### 緊急地震速報（警報）
```json
{
  "type": "emergency",
  "id": 20260417231454,
  "lines": [
    {
      "text": "<align=\"center\">【緊急地震情報（警報）】\n<align=\"left\">千葉県で地震 強い揺れに警戒\n<nobr>千葉</nobr>  <nobr>茨城</nobr>  <nobr>栃木</nobr>  <nobr>群馬</nobr>  <nobr>埼玉</nobr>  <nobr>東京</nobr>  <nobr>神奈川</nobr>  <nobr>静岡</nobr>",
      "duration": 15.0
    }
  ],
  "timestamp": "2026-04-02T09:30:00.123Z"
}
```
